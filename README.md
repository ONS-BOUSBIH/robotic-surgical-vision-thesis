# Robotic Surgical Vision Thesis: Segmentation and 3D Reconstruction of Robotic Surgical Instruments from Stereo Images

## Project Overview
This repository contains the code for my thesis project focused detection and 3D reconstruction of Da Vinci robot's surgical tools from stereo images.

## Project Structure
- `configs/`: Model configuration files (.yaml, .py).
- `src/`: Core Python modules for data processing and model wrappers.
- `notebooks/`: Research exploration and visualization.
- `utilities/`: helper utils scripts (logging, visualization,...)
